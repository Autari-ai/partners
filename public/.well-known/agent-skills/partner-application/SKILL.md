# Partner Application

Submit a partner application to join the Autari automation partnership programme.

## Endpoint

The partner application form is available at `https://partners.autari.co.uk/#apply`.

## Required Fields

| Field | Type | Description |
|-------|------|-------------|
| name | string | Full name of the applicant |
| email | string | Contact email address |
| expertise | string | Area of expertise (see options below) |

## Optional Fields

| Field | Type | Description |
|-------|------|-------------|
| phone | string | Contact phone number |
| company | string | Company or organisation name |
| heardAbout | string | How the applicant heard about the programme |
| message | string | Additional message or context |

## Expertise Options

- Finance / Accounting
- Marketing / Advertising
- Sales / CRM
- HR / Recruitment
- Legal
- Healthcare
- Property / Real Estate
- Architecture / Engineering
- Operations / Logistics
- Education / Training
- Other

## Response Timeline

Applications are reviewed within 48 hours. A short conversation follows to scope the first workflow.
